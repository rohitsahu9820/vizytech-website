# DataViz Pro — Company Website

A professional, multi-page website for a custom dashboard solutions company.

## Pages
- `index.html` — Homepage (Hero, Industries, Process, Portfolio, Testimonials)
- `pages/about.html` — About Us & Team
- `pages/services.html` — Services & Pricing
- `pages/portfolio.html` — Portfolio with Image Upload
- `pages/industries.html` — Industry-specific Solutions
- `pages/contact.html` — Contact Form

```
└── pages/
    ├── about.html
    ├── services.html
    ├── portfolio.html
    ├── industries.html
    └── contact.html
```

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g., `your-company-website`)
2. Upload all files maintaining the folder structure
3. Go to **Settings → Pages**
4. Set Source: **Deploy from a branch** → `main` → `/ (root)`
5. Save — your site will be live at `https://yourusername.github.io/your-company-website`
